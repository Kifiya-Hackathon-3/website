# Getting started

## Prerequisites

- Git
- Toolchains for the repos you use (e.g. Flutter/Dart for `mobile`, Node for `web`)

## Clone

```bash
git clone https://github.com/Kifiya-Hackathon-3/<repository>.git
cd <repository>
```

Replace `<repository>` with `mobile`, `web`, `liveness-api`, or `website` from the [organisation](https://github.com/Kifiya-Hackathon-3).

## Run (examples)

Commands depend on each repo’s README. Typical patterns:

```bash
# Node / web
npm install
npm run dev

# Flutter
flutter pub get
flutter run
```

## Download page

The public download landing page lives beside this docs app: open `index.html` under `eudi-download/` (or your deployed URL).
